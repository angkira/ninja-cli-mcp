# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/), and this project adheres to [Semantic Versioning](https://semver.org/).

## [Unreleased]

### Added
- Initial open source release of ninja-cli-mcp
- MCP stdio server delegating tasks to AI code CLI
- Support for Claude, GPT, Qwen, DeepSeek, Gemini, Llama models via OpenRouter
- Quick tasks, sequential plan execution, and parallel plan execution tools
- Comprehensive LiveBench notebooks with real task execution
- Scripts to register with Claude Code and Copilot CLI
- CI pipeline with linting, type checking, and tests across Python 3.11-3.13
- Contribution guidelines, code of conduct, and GitHub issue templates

## [0.1.0] - 2025-12-15

### Added
- Initial implementation with core MCP server and tool executor
- Ninja driver for subprocess management
- Instruction builder and scope enforcement
- Logging utilities and metrics
- Tests covering quick tasks, sequential and parallel plan execution
- README with architecture diagrams and usage instructions
